# Is It AI? - Content

Static content backend for the Is It AI? Android app, served via GitHub Pages.

**Live URL:** https://kevinrutledge.github.io/is-it-ai-content/

## Adding a New Pack

1. Create a directory: `packs/{pack-id}/images/`
2. Add optimized WebP images to the images/ directory
3. Create `packs/{pack-id}/manifest.json` with content items (see schema below)
4. Run `python3 generate_packs.py` to update packs.json
5. Run `python3 validate.py` to check for errors
6. Commit and push to main

## Image Requirements

- Format: WebP
- Max width: 1440px (aspect ratio preserved)
- Quality: 90
- Metadata: stripped (no EXIF)
- Convert with: `cwebp -q 90 -resize 1440 0 -metadata none input.jpg -o output.webp`

## Scripts

**generate_packs.py** - Regenerates packs.json from disk. Computes itemCount and sizeMb from actual files. Preserves human-written name and description fields.

**validate.py** - Pre-push validation. Checks that all manifests are valid JSON, all referenced images exist, no orphan images, no duplicate IDs, and packs.json matches the directory structure.

## Schemas

### packs.json
```json
[
  {
    "id": "pack-1",
    "name": "Display Name",
    "description": "Short description for the pack card",
    "itemCount": 12,
    "sizeMb": 2.2
  }
]
```
`itemCount` and `sizeMb` are auto-generated by `generate_packs.py`.

### manifest.json (per pack)
```json
[
  {
    "id": "unique_item_id",
    "filename": "images/example.webp",
    "type": "ai",
    "packageId": "pack-1",
    "explanation": "Why this is AI (for AI images)",
    "annotations": [
      {
        "x": 0.5,
        "y": 0.3,
        "radius": 0.08,
        "artifactType": "anatomical",
        "label": "Short label",
        "description": "What to look for"
      }
    ]
  },
  {
    "id": "another_item_id",
    "filename": "images/photo.webp",
    "type": "real",
    "packageId": "pack-1",
    "attribution": {
      "photographer": "Name",
      "year": 2024,
      "context": "Where/what"
    },
    "whyItLookedAI": "Why people thought it was fake",
    "story": "The full story behind the image"
  }
]
```

### Fields

| Field | Required | Description |
|-------|----------|-------------|
| id | yes | Unique identifier across all packs |
| filename | yes | Relative path to image |
| type | yes | "ai" or "real" |
| packageId | yes | Pack ID this item belongs to |
| explanation | AI only | Why the image is AI-generated |
| attribution | real only | Photographer name, year, context |
| whyItLookedAI | real only | Why the real photo seemed fake |
| story | optional | Narrative behind the image |
| annotations | optional | Array of artifact markers (AI images) |

### Annotation artifact types
- `anatomical` - body/hand errors (blue)
- `texture` - surface/text issues (red)
- `background` - scene errors (purple)
- `facial` - face artifacts (yellow)
- `lighting` - shadow/light issues (green)

Coordinates are normalized 0.0-1.0 relative to image dimensions.
